# Cronos Snapshots

To streamline the node setup process and reduce sync times, Cronos supports multiple snapshot and sync methods, both **officially provided** and supported by **ecosystem partners**. These options allow node operators to start from a recent blockchain state without syncing from block 1.

This section provides an overview of the various sync methods available and guidance on using each.\
